# GobyPoc



#### 命令规则

产品名-危害等级-CVE

例子：

- Gitlab_RCE_CVE_2021_22205
- 表示`Gitlab`产品`RCE`漏洞，`CVE`编号为`CVE_2021_22205`

#### CVE POC通知群

- 使用[github-cve-monitor](https://github.com/yhy0/github-cve-monitor)

- [点击加入tg通知群](https://t.me/+de2fxXndJfFjNTVl)
